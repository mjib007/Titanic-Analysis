## 📖 專案簡介
本專案是專為 **法律背景專業人士** 與 **高中生** 設計的數據科學入門教材。我們不從枯燥的程式語法開始，而是從經典的鐵達尼號（Titanic）海難事件出發，帶領學習者理解如何運用 **機器學習 (Machine Learning)** 從歷史數據中發現規律，並進行預測。

## 🎯 學習目標
1. **資料清洗 (Data Cleaning)**：學習如何像處理法律卷證一樣，將雜亂的原始數據整理成可分析的資訊。
2. **數據邏輯 (Data Logic)**：觀察性別、年齡、社會地位（艙等）如何影響生存機率。
3. **機器學習入門**：親手構建神經網路模型，預測經典電影角色（如 Jack 與 Rose）的生存命運。
4. **AI 倫理思辨**：討論模型背後的偏見問題，理解 AI 在法律與社會應用中的公平性挑戰。

## 🛠️ 技術工具

- **程式語言**：Python 3
- **核心套件**：Pandas (資料處理), Scikit-learn (機器學習), Keras/TensorFlow (神經網路)
